Some background: TigerPair an alumni matching platform that allows for various student groups to sign up as admins and maintain matching of students and alumni in their group. But instead of creating a different profile for other groups, students and alumni only need to create their profile once. They can then move between any student group they have permission to join.
The site is rough around the edges right now, so any feedback we get would be extremely helpful! The about text at the bottom is not the best, so please send questions in the below feedback form! Also, every user account (student, alumni, and admin) has a basic tutorial available at the bottom left corner of the page. Try those out if ur confused, let us know if they are really bad!

More details if you are interested:

Since we already did a few alumni tests, we have opened up the alumni side to students for testing (we can't get a ton of alumni testing it in this state). So when you are playing around with it, feel free to make alumni or student account to test!

If you are testing with friends, you can make an admin group account and have them sign up with that group id/password too. But if you want to sign up for an established admin account, just leave the group id section clear on the before we begin page. It will default you to group 0.

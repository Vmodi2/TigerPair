Welcome to TigerPair

TigerPair is a user-friendly alumni engagement platform that finds optimal matches based on features chosen by the mentorship project administrator. Using vectorization techniques on the cutting edge of machine learning, the algorithm can analyze keywords in the student and alumni profiles to find optimal matches based on major, degree type, general keyword interests, preferred degree of involvement, and any other feature that the admin desires to include.
